## üè• Medical Complex Website

This project is a responsive and multilingual medical website developed using pure HTML, CSS, and JavaScript.  
It includes modern UI features such as a hero slider, scroll-based animations, modal popups for specialties, embedded Google Maps, WhatsApp floating contact button, and a newsletter form.

The website supports both French (LTR) and Arabic (RTL) layouts, making it suitable for bilingual audiences.

### Features
- Responsive layout (mobile, tablet, desktop)
- Hero image slider
- Fade-up animations on scroll
- Modal popups for medical specialties
- Google Maps integration
- WhatsApp floating contact button
- Newsletter subscription form
- Arabic RTL and French LTR support

### Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)
- Font Awesome
